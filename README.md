# ModelBenchmarks

## Description

This repository holds the benchmarking of pretrained CNN models for identifying objects. The model is to identify common objects within a standard household environment. The model that is the most suitable for a Raspberry Pi 5 8GB embedded system will be transferred using ONNX Runtime.

## Pre-requisites

After cloning this repository, install all dependencies onto a virtual environment before running this program, use the command:
```pip3 install -r requirements.txt```

## Running

Simply run the code after cloning and installing dependencies, this program will use a CUDA supported GPU if available.
